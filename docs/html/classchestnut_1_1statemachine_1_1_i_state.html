Detailed Description

template<typename StateNameType, class ParentStatemachine>
class chestnut::statemachine::IState< StateNameType, ParentStatemachine >

Base generic abstract class used for creating state interface.

Template Parameters
StateNameTypea type of states' name
ParentStatemachinetype of the statemachine that will house the state

ParentStatemachine type should be forward declared before the state interface.

StateNameType can be any type you'd want to identify your states by, like an enum, string or a regular int. The type must however be able to be used for std::unordered_map as a key, so if you have a custom type you would want to use a name, you need to implement a std::hash template specialization for it
