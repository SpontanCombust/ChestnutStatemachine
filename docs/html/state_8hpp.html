◆ NULL_STATE

const std::type_index chestnut::fsm::NULL_STATE = std::type_index( typeid(nullptr) )

A constant type index meant to represent a lack of state.

There are currently 2 situations when NULL_STATE is used:

  1. When statemachine is being initialized with its init state, then NULL_STATE is passed to onEnter of that state
  2. When statemachine is being deleted, then NULL_STATE is passed to onExit of every state on the state stack before deleting them
